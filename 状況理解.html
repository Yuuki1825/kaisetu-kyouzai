【数学】状況理解とは

状況理解とは

状況理解の定義

「どれが自由に決まり、それにより他がどう一意に定まるのか」という問題の構造を理解することであり、言い換えると「その問題の自由度を求めること」である。

※自由度とは
→ 自由に値を決めることのできる変数の数

状況理解のツールには以下の2つがある。

  • 具体的な値の代入（実験）
  • 図形の成り立ちを理解しながらの状況図示

1　具体的な値の代入（実験）

以下の問題を例にとる。

出典：広島大

(1) \(a\) と \(b\) を互いに素な自然数とし、自然数 \(n\) に対し、

\[\frac{1}{n+1} < \frac{b}{a} < \frac{1}{n}\]

が成り立つとする。互いに素な自然数 \(c, d\) により

\[\frac{b}{a} - \frac{1}{n+1} = \frac{d}{c}\]

と表すとき、\(d < b\) となることを示せ。

問題文が、\(a, b, c, d, n\) で構成されており意味が分かりづらい。

そこで「具体的な値の代入」により どれを決めればどう他が一意に定まるのか を明らかにしていく。

何に具体的な値を代入すればいいか、一見分からないかもしれませんが、その場合は色々試してみる。

最終的には以下のような実験ができるとよい。

このように「\(a, b\) を決めたら他の値はすべて一意に定まる」ということが分かる。

これが状況理解を達成した状態である。

この時、この問題は 自由度 2 である。（\(a\) と \(b\) は自由に決めることができるが、\(a\) と \(b\) を決めたら他の値は一意に定まるから）

2　図形の成り立ちを理解しながらの状況図示

以下の問題を例にとる。

2015年 東大文系数学 第3問

\(\ell\) を座標平面上の原点を通り傾きが正の直線とする。さらに、以下の3条件 (i), (ii), (iii) で定まる円 \(C_1, C_2\) を考える。

(i) 円 \(C_1, C_2\) は2つの不等式 \(x \geq 0\), \(y \geq 0\) で定まる領域に含まれる。

(ii) 円 \(C_1, C_2\) は直線 \(\ell\) と同一点で接する。

(iii) 円 \(C_1\) は \(x\) 軸と点 \((1, 0)\) で接し、円 \(C_2\) は \(y\) 軸と接する。

円 \(C_1\) の半径を \(r_1\)、円 \(C_2\) の半径を \(r_2\) とする。\(8r_1 + 9r_2\) が最小となるような直線 \(\ell\) の方程式と、その最小値を求めよ。

このように、「どの点を決めればこの点が決まり、その点とこの点を結んだ直線とこの直線の交点により次の点が決まり、…」のように、各点や直線はどのように位置が定まっているかを理解することが必要になる。仮に図が与えられていたとしても自分で書き直し、図形の成り立ちを理解することが必要になる。

ちなみにこの問題の自由度は 自由度 1 となる。（自由に決められるのは直線 \(\ell\) の傾きのみ）

そのため、この問題は最終的には1変数関数（あるいは2変数関数の制約式1つ）などの最大最小になる帰着まで見えてくる。

平面上に三角形 \(ABC\) があり、\(CA = 1\), \(AB = \sqrt{5}\), \(BC = 2\) である。また、三角形 \(ABC\) と合同な三角形 \(DEF\) があり、\(FD = CA\), \(DE = AB\), \(EF = BC\) である。三角形 \(DEF\) が次の3つの条件 (i), (ii), (iii) を満たしながら動く。

(i) \(E\) は辺 \(AB\)（両端は除く）上にある。

(ii) \(C\) は辺 \(EF\)（両端は除く）上にある。

(iii) 辺 \(CA\) と辺 \(DE\) は交点をもつ。

辺 \(CA\) と辺 \(DE\) の交点を \(G\) とするとき、線分 \(AG\) の長さの最大値を求めよ。

第4問

\(f(x) = x^3 - 3x\) とする。\(-2 < p < q < 2\) を満たす実数 \(p, q\) に対して、実数 \(y\) の集合 \(I\) を

\[I = \{y \mid p \leq y \leq q\}\]

とする。さらに、\(f(x)\) が \(I\) に属するような実数 \(x\) の集合を \(D\) とする。つまり、

\[D = \{x \mid p \leq f(x) \leq q\}\]

と定める。

実数 \(p, q\) が \(-2 < p < q < 2\) を満たして動くとき、\(I\) の長さ \(l(I)\), \(D\) の長さ \(l(D)\) について、比 \(\displaystyle\frac{l(I)}{l(D)}\) のとりうる値の範囲を求めよ。

ただし、実数の集合 \(S\) が、共通部分のないいくつかの区間からなるとき、\(S\) の長さ \(l(S)\) は、各区間の長さの和とする。
例えば、\(S = \{x \mid 1 \leq x \leq 2\} \cup \{x \mid 3 \leq x \leq 5\}\) ならば、\(l(S) = (2 - 1) + (5 - 3) = 3\) である。
