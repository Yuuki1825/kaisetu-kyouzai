逆像法 超わかりやすく解説

松濤舎

この教材は 補助教材1 の続きとして、逆像法の考え方を「ゼロから腑に落とす」ことを目的に作っています。

「逆像法ってよく分からない」「やり方は知っているけど、なぜそうするのか説明できない」という方に向けて、具体例 → 一般化の順で丁寧に進めます。

1 イントロ — そもそも何を求めたいのか

まず次の超基本問題から始めましょう。

問題　\(x\) が全実数を動くとき、\(x^2 + 2x\) の最小値を求めよ。

多くの人が真っ先に思いつくのは「平方完成」だと思います。

\[x^2 + 2x = (x+1)^2 - 1\]

よってグラフから最小値は \(-1\)。これは正しい解答ですし、これが順像法です（後で詳しく扱います）。

ここでは、あえて別の角度から同じ問題を眺めてみます。

視点の切り替え

「最小値を求める」というお題は、言いかえれば

「\(x^2 + 2x\) が取りうる値の集合を求めて、その下限を読む」

ことに他なりません。

つまり、本当に欲しいのは「取りうる値の集合」なのです。

では「取りうる値の集合」をどうやって作るか？　もっとも原始的な方法は、数直線上の点を 1 個ずつ調べることです。

原始的なアプローチ：値を 1 個ずつ「OK か NG か」確かめていく

もちろん、無限個の点をすべて調べるのは不可能です。でも、原理は同じです。

この「点を 1 個ずつ調べる」という発想を貫いたとき、自然に出てくるのが 逆像法 です。

2 ウォームアップ「2 は取りうる？」を実際にやってみる

本当に点を 1 個ずつ調べる気持ちで、「\(x^2 + 2x\) は値 2 を取りうるか？」 を実際に判定してみます。

Q.「\(x^2 + 2x = 2\) を満たす実数 \(x\) は存在するか？」

これは 2 次方程式 \(x^2 + 2x - 2 = 0\) の実数解の有無を聞いているのと同じです。

判別式：\(\dfrac{D}{4} = 1^2 - 1\cdot(-2) = 3 > 0\)　→　実数解あり。

A.　よって \(x = -1 \pm \sqrt{3}\) が存在する。つまり 2 は取りうる。

続けて、「\(-2\) は取りうるか？」 も同じやり方で。

Q.「\(x^2 + 2x = -2\) を満たす実数 \(x\) は存在するか？」

2 次方程式 \(x^2 + 2x + 2 = 0\)。

判別式：\(\dfrac{D}{4} = 1 - 2 = -1 < 0\)　→　実数解なし。

A.　よって \(-2\) は取りえない。

2 個の値を判定できた

ここまでの 2 例で、判定の手順が完全に同じであることに注目してください。

逆像法の 2 ステップ

  1. ①具体的な値を例にして、「その値は取りうるか／取り得ないのか」という問いを判別式で結論づける。
  2. ②①と全く同じ手順を、ある定数 \(K\) に対して行う。

つまり ①で型を作り、②で型に \(K\) を流し込むだけです。次の章で②をやります。

3 核心 — 具体値を \(K\) に置き換えるだけ

無限個の値をひとつずつ調べる代わりに、「ある値 \(K\) が取りうるかどうか」 をまとめて判定します。

やることは 2 と全く同じ。

\[x^2 + 2x = K \quad \Longleftrightarrow \quad x^2 + 2x - K = 0\]

これを満たす実数 \(x\) が存在する条件は、判別式 \(\geqq 0\)：

\[\frac{D}{4} = 1^2 - 1\cdot(-K) = 1 + K \geqq 0 \quad \Longleftrightarrow \quad K \geqq -1\]

\(K\) を動かして判別式の値が変わる様子。\(K=-1\) を境に「取りうる／取り得ない」が切り替わる

取りうる値の集合は \(\{K \mid K \geqq -1\} = [-1, \infty)\)。よって最小値は \(\boldsymbol{-1}\)。

平方完成で出した答えと、完全に一致します。

なぜ判別式を使うのか？

キーになる同値変形は次の 1 行です。

「\(K\) は取りうる」 \(\Longleftrightarrow\) 「\(f(x) = K\) を満たす実数 \(x\) が存在する」

"取りうる" の言いかえが "実数 \(x\) の存在" であり、それを判定する道具のひとつが判別式です。

2 次方程式なら判別式、もう少し複雑なら領域図示やグラフの共有点に置き換わります（道具は問題ごとに変わります）。

逆像法の本質はただひとつ：
「具体値で立てた判定手順を、そのまま \(K\) に適用する」

4 順像法との対比 — 同じ問題を 2 通りで

3 章までやった \(x^2 + 2x\) の最小値を、順像法と逆像法で並べて比較します。

　 順像法（平方完成） 逆像法（K で存在判定）
考え方 各 \(x\) でどんな値になるかを順に追う 各値 \(K\) が取りうるかを逆向きに判定
式変形 \(x^2 + 2x = (x+1)^2 - 1\) \(x^2 + 2x - K = 0\) に対し \(\dfrac{D}{4} = 1 + K\)
判定 頂点 \((-1, -1)\) でグラフ最小 \(D \geqq 0\) すなわち \(K \geqq -1\)
最終形 最小値 \(-1\) 取りうる値の集合 \([-1, \infty)\) → 最小値 \(-1\)

結果は同じ。視点（時間の向き）が逆なだけです。

使い分けの目安

1 文字の関数なら、たいてい順像（平方完成・微分）が速い。
しかし条件付き 2 文字以上になると、平方完成や微分では押せなくなり、逆像法が威力を発揮します。次章以降の代表問題はまさにそのパターンです。

5 代表問題① 第1問：\(x^2 - xy + y^2 = 3\) における \(x\) の最大最小

第1問

実数 \(x, y\) が \(x^2 - xy + y^2 = 3\) を満たすとする。

(1) \(x\) の最大値と最小値を求めよ。

(2) \(2x + y\) の最大値と最小値を求めよ。

(1) \(x\) の最大最小

「順像で行く」と \(y\) を消そうとしますが、\(y^2 = 3 - x^2 + xy\) が \(y\) について解けず、平方完成も微分もうまく行きません。
こういうときは 「順像が無理なら逆像」とマインドを切り替えます。

ステップ① まずは具体値で

「\(x = 1\) は取りうるか？」

\(x = 1\) を代入すると \(1 - y + y^2 = 3\)、すなわち \(y^2 - y - 2 = 0\)。

判別式 \(D = 1 + 8 = 9 > 0\)　→　実数 \(y\) が存在。

A. よって \(x = 1\) は取りうる。

※ ここで因数分解できる手前で「具体値の代入で実験する → 判別式で判定」という形を意識してください。次のステップで同じ形を使います。

ステップ② \(K\) で同じ手順を

「\(x = K\) は取りうるか？」

\(x = K\) を代入すると

\[K^2 - Ky + y^2 = 3 \quad \Longleftrightarrow \quad y^2 - Ky + (K^2 - 3) = 0\]

これは \(y\) についての 2 次方程式です（\(K\) はパラメータとみなす）。実数 \(y\) が存在する条件は

\[D = K^2 - 4(K^2 - 3) = -3K^2 + 12 \geqq 0\]

\[\Longleftrightarrow K^2 \leqq 4 \Longleftrightarrow -2 \leqq K \leqq 2\]

取りうる値の集合は \([-2, 2]\) なので、\(x\) の最大値 2、最小値 \(-2\)。

楕円 \(x^2-xy+y^2=3\) と縦線 \(x=K\) の共有点。共有点があるのは \(-2 \leqq K \leqq 2\)

図形的な裏付け

条件式 \(x^2 - xy + y^2 = 3\) は実は 45° 傾いた楕円。
「\(x = K\) は取りうる ⇔ 縦線 \(x = K\) と楕円が共有点を持つ」と読みかえれば、判別式で求めた範囲が楕円の \(x\) 座標の最大最小に一致することが視覚的にも納得できます。

(2) \(2x + y\) の最大最小

同じ発想で「\(2x + y = K\) は取りうるか？」を考えます。
\(2x + y = K\) は1 次の等式なので、まずこれで \(y\) を消去できます（鉄則：等式条件は 1 文字消去）。

\(y = K - 2x\) を条件式に代入：

\[x^2 - x(K - 2x) + (K - 2x)^2 = 3\]

\[\Longleftrightarrow x^2 - Kx + 2x^2 + K^2 - 4Kx + 4x^2 = 3\]

\[\Longleftrightarrow 7x^2 - 5Kx + (K^2 - 3) = 0\]

これを満たす実数 \(x\) が存在する条件は

\[D = 25K^2 - 28(K^2 - 3) \geqq 0\]

\[\Longleftrightarrow -3K^2 + 84 \geqq 0 \Longleftrightarrow K^2 \leqq 28\]

よって \(-2\sqrt{7} \leqq K \leqq 2\sqrt{7}\)。最大 \(2\sqrt{7}\)、最小 \(-2\sqrt{7}\)。

(1) も (2) も判別式に流し込むだけのテンプレ操作だけで解けました。これが「条件付き 2 変数の最大最小」での逆像法の威力です。

6 代表問題② 第3問：空間ベクトルだって逆像法

第3問

\(xyz\) 空間の中で、\((0, 0, 1)\) を中心とする半径1の球面 \(S\) を考える。点 \(Q\) が \((0, 0, 2)\) 以外の \(S\) 上の点を動くとき、点 \(Q\) と点 \(P(1, 0, 2)\) の2点を通る直線 \(l\) と平面 \(z = 0\) との交点を \(R\) とおく。\(R\) の動く範囲を求め、図示せよ。

赤本などの解答では「空間ベクトルの問題」として処理されますが、背景には逆像の考えが流れています。

ステップ① まずは具体点で「\(R = (2, 3, 0)\) は通るか？」

もし点 \(R = (2, 3, 0)\) が \(R\) の動く範囲に含まれるなら、「\(P(1,0,2)\) と \(R(2,3,0)\) を結ぶ直線が球面 \(S\) と \((0,0,2)\) 以外で共有点を持つ」はず。

そこで直線をパラメータ \(t\) で表します：

\[\vec{P} + t(\vec{R} - \vec{P}) = (1, 0, 2) + t(1, 3, -2) = (1+t,\; 3t,\; 2-2t)\]

これが球面 \(S\)（中心 \((0,0,1)\)、半径 1）上にある条件は

\[(1+t)^2 + (3t)^2 + (2-2t-1)^2 = 1\]

\[\Longleftrightarrow (1+t)^2 + 9t^2 + (1-2t)^2 = 1\]

\[\Longleftrightarrow 14t^2 - 2t + 1 = 0\]

これを満たす実数 \(t\) が存在するかは、判別式：

\[\frac{D}{4} = 1 - 14 = -13 < 0\]

よって \(R = (2, 3, 0)\) は通らない。

具体点 \(R\) について「\(P\) と結んだ直線が球と交わるか」を判別式で判定する

ステップ② 一般点 \(R = (X, Y, 0)\) で同じ手順を

具体点でやった判定の手順をそのままコピーして、\(R = (X, Y, 0)\) に適用すればよい。

\[\vec{P} + t(\vec{R} - \vec{P}) = (1 + t(X-1),\; tY,\; 2 - 2t)\]

球面上の条件：

\[(1 + t(X-1))^2 + (tY)^2 + (1 - 2t)^2 = 1\]

これを展開して \(t\) について整理し、判別式 \(\geqq 0\) かつ \(t \neq\) (除外する \((0,0,2)\) に対応する \(t\) 値) を求めれば、\(R\) の動く範囲が出ます。

具体点 \((2,3,0)\) でやった「直線 → 球の方程式に代入 → \(t\) の判別式」という手順を、
そのまま \((X,Y,0)\) に流し込むのが逆像法。

逆像法を知らないと、この問題は「いきなり \(R\) を文字で置いて計算を始める」ように見え、なぜそうするのか分からないまま手順だけ覚えることになります。
逆像で考えれば「具体点で実験 → 一般化」というごく自然な流れに見えてくるはずです。

7 代表問題③ 第5問：放物線の通過領域（誘導の利用）

第5問

\(f(x) = x^2 - px + q\) とする。ここで \(p, q\) は実数とする。\(p, q\) は次の条件を満たして動く。

【条件】2次方程式 \(f(x) = 0\) は実数解をもち、2解の和の絶対値と2解の積の絶対値は等しい

(1) 点 \((p, q)\) が存在する範囲を \(pq\) 平面に図示せよ。

(2) 放物線 \(y = f(x)\) が通りえない範囲を \(xy\) 平面に図示せよ。

まず条件を式に直します。判別式 \(p^2 - 4q \geqq 0\)、解と係数の関係から「和の絶対値 \(=|p|\)」「積の絶対値 \(= |q|\)」なので

\[p^2 - 4q \geqq 0 \quad \text{かつ} \quad |p| = |q|\]

これが (1) の答えとなる \(pq\) 平面の領域です。

(2) の核心は「点 \((X, Y)\) を通る \((p, q)\) が存在するか」

放物線 \(y = x^2 - px + q\) が点 \((X, Y)\) を通る条件は

\[Y = X^2 - pX + q \quad \Longleftrightarrow \quad q = pX + Y - X^2\]

これは \(pq\) 平面における直線です（\(X, Y\) は固定の定数とみなす）。

逆像法 ＋ 誘導の利用

(2) で問われているのは 「\((X, Y)\) を通る \((p, q)\) が条件下に存在するか」。
ここで (1) で求めた領域がまさに「条件を満たす \((p, q)\) の集合」になっているので、

「(X, Y) を通る ⇔ 直線 \(q = pX + Y - X^2\) が (1) の領域と共有点を持つ」

と読みかえられます。
よって (1) で求めた領域に直線を重ねる図を描けば、(2) の答えが出る、というのが「誘導の利用」の意味です。

計算は補助教材1の page08 にあるとおり、通り得ない範囲は

\[\left\{(X, Y) \;\middle|\; Y < X^2 - 1 \;\text{かつ}\; Y < (X-2)^2 \;\text{かつ}\; Y > -X^2-1 \;\text{かつ}\; \cdots \right\}\]

のように複数の条件の組み合わせで表現され、\(-1 < x < 1\) 部分の面積は 4。

※詳細な場合分けは補助教材1の問題を実際に解いてみてください。キモは「逆像法の存在条件 → 誘導の利用」という流れです。

8 逆像法の本質と適用範囲

逆像法の本質

逆像法の本質は「具体値を入れたら、その値が取りうるかどうかをどう判断するか」を掴むことにあります。

判定の道具は問題によって様々です（判別式、領域、グラフの共有点、ベクトルの長さ条件、…）。
だから一般化はできないし、しない方がいい。問題ごとに「具体値で実験 → 同じ手順を \(K\) や \((X, Y)\) に適用」を貫くのが大事です。

逆像法が活きる問題ジャンル

  • 軌跡領域問題　…　「点 \((X, Y)\) が軌跡上にある ⇔ 条件を満たすパラメータが存在する」
  • 通過領域問題　…　「曲線族が通る ⇔ パラメータの存在条件」
  • 条件付きの最大最小　…　「\(K\) が取りうる ⇔ 条件を満たす変数組が存在する」

使う／使わないの判断目安

問題の構造 第一選択
パラメータ 1 種・3 次以下 順像法（微分）でも逆像法でも OK。やりやすい方を
パラメータ 2 種・対称式 順像（実数条件）／逆像どちらも OK
条件付き 2 変数で 1 文字消去できない 逆像法が第一選択
「軌跡上の点を通るか」を問われる空間ベクトル 逆像法（具体点 → 一般点）
通過領域・誘導付き 逆像法＋誘導の利用
迷ったら：
「順像が無理なら逆像を試す」

9 自分で解いてみよう

補助教材1 の 第2問・第4問は、ここまでの考え方を自分で適用する練習問題です。ヒントだけ載せておきます。

第2問

(1) \(x = s + t + 1,\; y = s - t - 1\) （\(s, t \geqq 0\)）の点 \((x, y)\) の動く範囲を図示せよ。

(2) \(x = st + s - t + 1,\; y = s + t - 1\) （\(s, t\) 実数）の点 \((x, y)\) の動く範囲を図示せよ。

ヒント　「点 \((X, Y)\) が動く範囲に入る ⇔ \(\begin{cases}X = s + t + 1\\ Y = s - t - 1\end{cases}\) を満たす \(s, t\;(\geqq 0)\) が存在する」と読みかえます。
(1) は \(s, t\) について解いて \(s, t \geqq 0\) を \(X, Y\) の不等式に翻訳。
(2) は \(s + t = Y + 1,\; st = X - Y - 2\) のように対称式に持ち込み、解と係数の関係から実数条件（判別式）に帰着。

第4問

実数 \(a, b, c\) が \(a + b + c = 8,\; a^2 + b^2 + c^2 = 32\) を満たすとき、実数 \(c\) の最大値を求めよ。

ヒント　「\(c = K\) は取りうるか？」を考えます。\(c = K\) を代入すると \(a + b = 8 - K,\; a^2 + b^2 = 32 - K^2\)。
\(a, b\) は 2 次方程式 \(t^2 - (8-K)t + \dfrac{(8-K)^2 - (32-K^2)}{2} = 0\) の解。
これを満たす実数 \(a, b\) が存在する ⇔ 判別式 \(\geqq 0\)。あとは \(K\) の不等式を解くだけ。

おわりに

逆像法は「点を 1 個ずつ調べたい」という素朴な気持ちを、判別式や領域図示の道具を借りて実現する手法です。

解答だけ眺めると魔法のように見えますが、根本は 「具体値で実験 → 同じ手順を \(K\) に流し込む」 だけ。
これからは「この求め方は逆像だと思えば当たり前だな」と思えるようになることを目指してください。
