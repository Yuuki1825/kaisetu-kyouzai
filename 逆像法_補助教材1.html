1 逆像法とは

逆像法

逆像法は、「この点はOKですか?」と、1つ1つ点を調べていき、何百の点を調べていくことで取りうる範囲が見えてくるという考えが根本にあります。

しかし、座標平面上（あるいは数直線上）のすべての点を調べるのは不可能なので、\((X, Y)\) や \(k\) のように一般化した定数で同じ議論をします。

この考え方が逆像法です。軌跡領域問題や最大最小で使われます。

2 順像法とは

順像法についてはいろいろな説明の仕方がされますが、端的に言えば、逆像法の対になる考えです。

自然なプロセスで"演繹的に"範囲を求めていく考え方を順像法と呼んでいます。

2次関数の最大最小で「平方完成する」のも、3次関数の問題で「微分してグラフを描いて求める」のも順像法です。

もちろん2変数関数の場合は演繹的に解く方法が「1文字固定」なので、多くの参考書では「順像法＝1文字固定」と紹介されますが、順像法はもっと意味が広く、逆像法以外の求め方はすべて順像法だと思っています。

（ここまで書きましたが、逆像が分かっているならば順像の「概念」(\(\neq\) 解法) は理解できなくてもいいです）

3 逆像法5選

第1問

実数 \(x, y\) が \(x^2 - xy + y^2 = 3\) を満たすとする。

(1) \(x\) の最大値と最小値を求めよ。

(2) \(2x + y\) の最大値と最小値を求めよ。

第2問

(1) \(x = s + t + 1,\; y = s - t - 1\) とおく。\(s, t\) が \(s \geq 0,\; t \geq 0\) の範囲を動くとき、点 \((x, y)\) の動く範囲を図示せよ。

(2) \(x = st + s - t + 1,\; y = s + t - 1\) とおく。\(s, t\) が実数全体を動くとき、点 \((x, y)\) の動く範囲を図示せよ。

第3問

\(xyz\) 空間の中で、\((0, 0, 1)\) を中心とする半径1の球面 \(S\) を考える。点 \(Q\) が \((0, 0, 2)\) 以外の \(S\) 上の点を動くとき、点 \(Q\) と点 \(P(1, 0, 2)\) の2点を通る直線 \(l\) と平面 \(z = 0\) との交点を \(R\) とおく。\(R\) の動く範囲を求め、図示せよ。

第4問

実数 \(a, b, c\) が \(a + b + c = 8,\; a^2 + b^2 + c^2 = 32\) を満たすとき、実数 \(c\) の最大値を求めよ。

第5問

\(f(x) = x^2 - px + q\) とする。ここで \(p, q\) は実数とする。\(p, q\) は次の条件を満たして動く。

【条件】2次方程式 \(f(x) = 0\) は実数解をもち、2解の和の絶対値と2解の積の絶対値は等しい

(1) 点 \((p, q)\) が存在する範囲を \(pq\) 平面に図示せよ。

(2) 放物線 \(y = f(x)\) が通りえない範囲を \(xy\) 平面に図示し、そのうち \(-1 < x < 1\) を満たす部分の面積を求めよ。

4 おわりに

他にも逆像法の問題は数えきれないほどありますが、参考書などの解答ではうまく式変形をして求めた感が満載で、背景に逆像の考え方が書かれていないことがほとんどです。

ですので、これからは「この求め方は逆像だと思えば当たり前だな」と思えるようになることを目指してほしいと思います。
