例題 81条件つき2変数関数(1)＊＊＊＊

実数 \(x, y\) の間に \(3x + y = 6\) という関係があるとき、次の問いに答えよ。

(1) \(z = x^2 + y^2\) の最小値と、そのときの \(x, y\) の値を求めよ。

(2) \(x \geqq 0, y \geqq 0\) のとき、\(x^2 + y^2\) のとりうる値の範囲を求めよ。

解答
定石①：文字消去で1変数に

\(3x + y = 6\) より \(y = 6 - 3x\) を代入する。

(1) \(z = x^2 + (6-3x)^2 = x^2 + 36 - 36x + 9x^2 = 10x^2 - 36x + 36\)

\(= 10\left(x - \displaystyle\frac{9}{5}\right)^2 + \displaystyle\frac{18}{5}\)

よって最小値 \(\displaystyle\frac{18}{5}\)（\(x = \displaystyle\frac{9}{5},\; y = 6 - \displaystyle\frac{27}{5} = \displaystyle\frac{3}{5}\)）

(2) \(x \geqq 0, y \geqq 0\) より \(0 \leqq x \leqq 2\) の制約が加わる。

\(z = 10\left(x - \displaystyle\frac{9}{5}\right)^2 + \displaystyle\frac{18}{5}\) は \(x = \displaystyle\frac{9}{5}\) で最小値 \(\displaystyle\frac{18}{5}\)、\(x = 0\) で \(z = 36\)、\(x = 2\) で \(z = 4\)。

よって \(\displaystyle\frac{18}{5} \leqq z \leqq 36\)

例題 82条件つき2変数関数(2)＊＊＊＊

実数 \(x, y\) の間に \(x^2 + 2y^2 = 4\) という関係があるとき、\(x + y^2\) の最大値、最小値と、そのときの \(x, y\) の値を求めよ。

解答
定石①：文字消去で1変数に

\(x^2 + 2y^2 = 4\) より \(y^2 = \dfrac{4 - x^2}{2}\) を代入する。

\(x + y^2 = x + \dfrac{4 - x^2}{2} = -\dfrac{x^2}{2} + x + 2 = -\dfrac{1}{2}(x-1)^2 + \dfrac{5}{2}\)

変域の確認：\(x^2 \leqq 4\) かつ \(y^2 \geqq 0\) より \(0 \leqq \dfrac{4-x^2}{2}\) すなわち \(-2 \leqq x \leqq 2\)。

\(x = 1\) で最大値 \(\dfrac{5}{2}\)（このとき \(y^2 = \dfrac{3}{2}\)、\(y = \pm\dfrac{\sqrt{6}}{2}\)）

\(x = -2\) で最小値 \(-2\)（このとき \(y^2 = 0\)、\(y = 0\)）

例題 115領域と最大・最小(2)＊＊＊＊

連立不等式 \(x \geqq 0, y \geqq 0, 4 \leqq x^2 + y^2 \leqq 9\) の表す領域において、\(x + 3y\) の最大値、最小値と、そのときの \(x, y\) の値を求めよ。

（大阪電気通信大・改）

解答
定石②：存在条件（線形計画法）

\(x + 3y = k\) とおき、直線 \(y = \dfrac{-x+k}{3}\) が領域と共有点を持つ条件を求める。

最大値：直線が大きい円 \(x^2 + y^2 = 9\) に接するとき。原点から直線 \(x + 3y = k\) までの距離が 3 に等しい。

\(\dfrac{|k|}{\sqrt{1^2 + 3^2}} = 3 \Rightarrow \dfrac{|k|}{\sqrt{10}} = 3 \Rightarrow k = 3\sqrt{10}\)（\(k > 0\) のとき）

接点：\(x + 3y = 3\sqrt{10}\) と \(x^2 + y^2 = 9\) を連立して \(x = \dfrac{3}{\sqrt{10}},\; y = \dfrac{9}{\sqrt{10}}\)（ともに正、条件を満たす）

最小値：領域の境界 \(y = 0,\; x = 2\) で \(k = 2\)、\(y = 0,\; x = 3\) で \(k = 3\)、\(x = 0,\; y = 2\) で \(k = 6\)、\(x = 0,\; y = 3\) で \(k = 9\) を比較すると、最小は \(x = 2, y = 0\) で \(k = 2\)。

小さい円 \(x^2 + y^2 = 4\) への接線では \(k = 2\sqrt{10} \approx 6.3\) となり、2 より大きい。したがって最小値は頂点 \((2, 0)\) で達成される。

最大値 \(3\sqrt{10}\)（\(x = \dfrac{3}{\sqrt{10}},\; y = \dfrac{9}{\sqrt{10}}\) のとき）、最小値 \(2\)（\(x = 2,\; y = 0\) のとき）

実数 \(x, y\) について、\(z = x^2 - 2xy + 2y^2 - 4x + 2y + 8\) の最小値と、そのときの \(x, y\) の値を求めよ。

解答
定石④：文字固定（予選決勝法）

\(y\) を固定して \(x\) について整理する：

\(z = x^2 + (-2y-4)x + (2y^2 + 2y + 8)\)

\(x\) について平方完成：\(z = (x - (y+2))^2 - (y+2)^2 + 2y^2 + 2y + 8\)

\(= (x-(y+2))^2 + y^2 - 2y + 4 = (x-(y+2))^2 + (y-1)^2 + 3\)

\(x = y+2,\; y = 1\) のとき最小値 \(3\)（すなわち \(x = 3,\; y = 1\)）。

最小値 3（\(x = 3,\; y = 1\) のとき）

4・8

実数 \(x\) および \(y\) について、\(-10 \leqq x \leqq 10\), \(-13 \leqq y \leqq 13\) であるとき、\(x^2 - 7x - y + 12\) の最大値は□である。また、\(xy^2 + 7xy + 11y^2 + 77y + 10x + 110\) の最小値は□である。

（14 武庫川女大・建築、薬）

解答
定石④：文字固定（予選決勝法）

第1式の最大値：\(f(x,y) = x^2 - 7x - y + 12\)

\(y\) を固定すると \(f\) は \(x\) の二次関数で、\(-10 \leqq x \leqq 10\) での最大は端点で達成される。\(x = -10\) のとき \(f = 100 + 70 - y + 12 = 182 - y\)、\(x = 10\) のとき \(f = 100 - 70 - y + 12 = 42 - y\)。端点では \(x = -10\) の方が大きい。

\(182 - y\) を \(-13 \leqq y \leqq 13\) で最大化すると \(y = -13\) のとき \(\mathbf{195}\)。

第2式の最小値：\(g(x,y) = xy^2 + 7xy + 11y^2 + 77y + 10x + 110\)

\(x\) について整理：\(g = (y^2 + 7y + 10)x + 11y^2 + 77y + 110\)

因数分解：\(y^2 + 7y + 10 = (y+2)(y+5)\)、\(11y^2 + 77y + 110 = 11(y+2)(y+5)\)

\(g = (y+2)(y+5) \cdot x + 11(y+2)(y+5) = (y+2)(y+5)(x+11)\)

\(-10 \leqq x \leqq 10\) より \(x + 11 \geqq 1 > 0\)。\((y+2)(y+5)\) の符号と \((x+11)\) の値を考えると、最小値は \((y+2)(y+5) = 0\) のとき \(g = 0\)（例：\(y = -2\) または \(y = -5\)）。

\((y+2)(y+5) < 0\)（\(-5 < y < -2\)）と \(x+11\) が最大（\(x=10\)）でも \(g = (y+2)(y+5) \cdot 21\)、最小は \(y = -3.5\) 付近で \((-1.5)(1.5)(21) = -47.25\) 程度となるが、これが最小かを確認すると \(y=-3.5, x=10\) では \(g = (-1.5)(1.5)(21) = -47.25\)。

ただし整数点や端点を確認：\(y = -5, x = -10\)：\(g = 0\)。\(y = -13, x = 10\)：\(g = (-11)(-8)(21) = 1848 > 0\)。

最大値 195（\(x = -10,\; y = -13\) のとき）、最小値 \(-\displaystyle\frac{189}{4}\)（\(x = 10,\; y = -\displaystyle\frac{7}{2}\) のとき）

例題 150三角関数の最大・最小(7)＊＊＊＊

実数 \(x, y\) が \(x^2 + y^2 = 4\)（\(x \geqq 0, y \geqq 0\)）を満たすとき、\(2x^2 + 3xy - y^2\) の最大値と最小値を求めよ。

解答
定石⑤：置換（三角関数パラメータ）

\(x = 2\cos\theta,\; y = 2\sin\theta\)（\(0 \leqq \theta \leqq \dfrac{\pi}{2}\)）とおく。

\(2x^2 + 3xy - y^2 = 2 \cdot 4\cos^2\theta + 3 \cdot 4\sin\theta\cos\theta - 4\sin^2\theta\)

\(= 8\cos^2\theta + 12\sin\theta\cos\theta - 4\sin^2\theta\)

\(= 4(1 + \cos 2\theta) + 6\sin 2\theta - 2(1 - \cos 2\theta)\)

\(= 6\cos 2\theta + 6\sin 2\theta + 2 = 6\sqrt{2}\sin\!\left(2\theta + \dfrac{\pi}{4}\right) + 2\)

\(0 \leqq \theta \leqq \dfrac{\pi}{2}\) より \(\dfrac{\pi}{4} \leqq 2\theta + \dfrac{\pi}{4} \leqq \dfrac{5\pi}{4}\)

この範囲で \(\sin\) の最大は \(2\theta + \dfrac{\pi}{4} = \dfrac{\pi}{2}\)（\(\theta = \dfrac{\pi}{8}\)）のとき \(1\)、最小は \(2\theta + \dfrac{\pi}{4} = \dfrac{5\pi}{4}\)（\(\theta = \dfrac{\pi}{2}\)）のとき \(-\dfrac{\sqrt{2}}{2}\)。

最大値 \(6\sqrt{2} + 2\)（\(\theta = \dfrac{\pi}{8}\) のとき）、最小値 \(6\sqrt{2} \cdot \left(-\dfrac{\sqrt{2}}{2}\right) + 2 = -6 + 2 = -4\)（\(\theta = \dfrac{\pi}{2}\)、すなわち \(x=0, y=2\) のとき）

例題 28コーシー・シュワルツの不等式＊＊＊＊

次の問いに答えよ。

(1) 不等式 \((a^2 + b^2)(c^2 + d^2) \geqq (ac + bd)^2\) を証明せよ。

(2) (1)の結果を利用して、不等式 \(5x^2 + 5y^2 \geqq (2x + y)^2\) を証明せよ。

解答
定石⑥：コーシー・シュワルツの不等式

(1) \((a^2+b^2)(c^2+d^2) - (ac+bd)^2\)

\(= a^2c^2 + a^2d^2 + b^2c^2 + b^2d^2 - a^2c^2 - 2abcd - b^2d^2\)

\(= a^2d^2 - 2abcd + b^2c^2 = (ad - bc)^2 \geqq 0\)

よって \((a^2+b^2)(c^2+d^2) \geqq (ac+bd)^2\)（等号成立：\(ad = bc\) のとき）

(2) (1)で \(a = 2,\; b = 1,\; c = x,\; d = y\) とおくと：

\((2^2 + 1^2)(x^2 + y^2) \geqq (2x + y)^2\)

背理法による証明と対偶による証明の違い
命題 \(p \Rightarrow q\) について、背理法では「\(p\) であって \(q\) でない」（命題が成り立たない）として矛盾を導くが、結論の「\(q\) でない」に対する矛盾でも、仮定の「\(p\) である」に対する矛盾でもどちらでもよい。後者の場合、「\(\bar{q} \Rightarrow \bar{p}\)」つまり対偶が真であることを示したことになる。このように考えると、背理法による証明と対偶による証明は似ているように感じられるが、本質的には異なるものである。対偶による証明は「\(\bar{q} \Rightarrow \bar{p}\)」を示す、つまり、（証明を始める段階で）導く結論が \(\bar{p}\) とはっきりしている。これに対し、背理法の場合、「\(p\) であって \(q\) でない」として矛盾が生じることを示す、つまり、（証明を始める段階では）どういった矛盾が生じるのかははっきりしていない。

この問題に関しても「背理法の問題」とできるので定石④が使えるものとします。

(2) (1)で6個の円を次々に描いていった。一般に \(n \geqq 2\) に対して \(3n\) 個の円を用いて同様の操作を行うとき、描かれる円の面積の総和 \(S_n\) と、それらの円の円周の長さの総和 \(C_n\) を求めよ。

(3) 数列 \(S_2, S_3, S_4, \cdots\) の極限値を求めよ。

解答
典型的な極限：自分で変数（半径 \(r_n\)）を導入し漸化式を立てる

準備：半径 \(r\) の円に内接する \(3n\) 個の等円の半径

各小円の中心は大円の中心から距離 \(r - \rho\) の位置に \(\dfrac{2\pi}{3n}\) 間隔で並ぶ（\(\rho\) は小円の半径）。隣り合う小円が外接する条件：

\[2\rho = 2(r-\rho)\sin\frac{\pi}{3n} \implies \rho = \frac{r\sin(\pi/(3n))}{1+\sin(\pi/(3n))}\]

(1) \(n=2\)（6個）のとき

\(\sin\dfrac{\pi}{6} = \dfrac{1}{2}\) より \(\rho_1 = \dfrac{1/2}{1+1/2} = \dfrac{1}{3}\)（大円半径 \(r=1\)）。

次の世代の小円は半径 \(\rho_1\) の円に内接する 6 個、その半径は \(\rho_2 = \dfrac{1}{3}\rho_1 = \dfrac{1}{9}\)。一般に \(\rho_k = \left(\dfrac{1}{3}\right)^k\)。

\[S_2 = \sum_{k=1}^{\infty} 6\pi\rho_k^2 = 6\pi\sum_{k=1}^{\infty}\frac{1}{9^k} = 6\pi \cdot \frac{1/9}{1-1/9} = \frac{6\pi}{8} = \frac{3\pi}{4}\] \[C_2 = \sum_{k=1}^{\infty} 6 \cdot 2\pi\rho_k = 12\pi\sum_{k=1}^{\infty}\frac{1}{3^k} = 12\pi \cdot \frac{1/3}{1-1/3} = 6\pi\]

(2) 一般の \(n \geqq 2\) のとき

\(s_n = \sin\dfrac{\pi}{3n}\) とおくと、初代の半径 \(\rho = \dfrac{s_n}{1+s_n}\)、縮小比は同じく \(\dfrac{s_n}{1+s_n}\)（相似）。

初代に \(3n\) 個あり、各世代で \(3n\) 倍されながら縮小比 \(q = \dfrac{s_n}{1+s_n}\) で続く：

\[S_n = \frac{3n\pi q^2}{1-q^2}, \qquad C_n = \frac{6n\pi q}{1-q}\]

（\(q = \dfrac{s_n}{1+s_n}\) より \(1-q = \dfrac{1}{1+s_n}\)、\(q^2 = \dfrac{s_n^2}{(1+s_n)^2}\)）

\[S_n = \frac{3n\pi s_n^2/(1+s_n)^2}{1-s_n^2/(1+s_n)^2} = \frac{3n\pi s_n^2}{(1+s_n)^2 - s_n^2} = \frac{3n\pi s_n^2}{1+2s_n}\] \[C_n = \frac{6n\pi s_n/(1+s_n)}{1/(1+s_n)} = 6n\pi s_n = 6n\pi\sin\frac{\pi}{3n}\]

(3) \(n \to \infty\) のとき

\(s_n = \sin\dfrac{\pi}{3n} \sim \dfrac{\pi}{3n}\)（\(n \to \infty\)）なので：

\[S_n = \frac{3n\pi s_n^2}{1+2s_n} \approx \frac{3n\pi(\pi/(3n))^2}{1} = \frac{3n\pi \cdot \pi^2/(9n^2)}{1} = \frac{\pi^3}{3n} \to 0\]

より正確には \(n\sin\dfrac{\pi}{3n} \to \dfrac{\pi}{3}\)、\(n s_n^2 = n \cdot \left(\dfrac{\pi}{3n}\right)^2 \cdot \left(\dfrac{\sin(\pi/(3n))}{\pi/(3n)}\right)^2 \to \dfrac{\pi^2}{9n} \to 0\)。

\[\lim_{n\to\infty} S_n = 0\]
